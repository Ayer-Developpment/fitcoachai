❓ Questions fréquentes

Comment annuler mon abonnement ?
Réglages iOS → Apple ID → Abonnements → FitCoach AI → Annuler.

Le scan code-barres ne trouve pas mon produit ?
La base Open Food Facts contient 3M+ produits. Si le tien est absent, ajoute les infos manuellement dans le chat coach.

L'IA ne répond pas ?
Vérifie ta connexion. Ferme et relance l'app. Si ça persiste, contacte-nous.

Comment restaurer mes achats ?
Réglages de l'app → Restaurer mes achats. Utilise le même Apple ID.

🔧 Signaler un bug

Email : support.fitcoachai@gmail.com avec description + version iOS.
